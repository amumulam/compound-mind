# Decision Logging Module

## 功能

记录重要决策，保持可追溯性。

## 决策记录格式

```
# 决策：[标题]

**日期**: YYYY-MM-DD
**背景**: [为什么需要做决策]
**选项**: 
1. [选项 1]
2. [选项 2]
**决策**: [选择了哪个]
**理由**: [为什么选这个]
**影响**: [这个决策会带来什么]
```

## 触发方式

在检查点提取时自动识别决策并记录。

## 存储位置

`life/decisions/YYYY-MM-DD-[slug].md`
