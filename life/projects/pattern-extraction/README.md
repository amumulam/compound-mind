# Pattern Extraction Module

## 功能

每周分析日志，发现主题和规律。

## 触发方式

通过 OpenClaw 的 cron 定时任务，每周日触发。

## 提取规则

### 高频问题
- 统计日志中出现多次的问题
- 识别"我总是在这里踩坑"

### 常用解决方案
- 统计成功解决问题的模式
- 形成可复用的解决方案模板

### 行为模式
- 时间分布（什么时候最活跃）
- 任务类型分布
- 情绪/状态变化

### 知识缺口
- 识别"我不知道 X"的情况
- 标记需要学习的领域

## 输出格式

```
## 周报 YYYY-WW

### 高频问题
- [问题]: 出现 N 次

### 常用解决方案
- [方案]: 使用 N 次

### 行为模式
- 最活跃时间: ...
- 主要任务类型: ...

### 知识缺口
- [缺口]: 需要学习

### 建议
- [改进建议]
```
