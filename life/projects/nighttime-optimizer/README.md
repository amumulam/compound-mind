# Nighttime Optimizer Module

## 功能

自动维护系统健康，保持整洁有序。

## 触发方式

每周日凌晨运行。

## 检查项目

### 1. 文件健康检查
- MEMORY.md 大小：超过 500 行警告，超过 1000 行必须拆分
- 日志文件数量：超过 30 天归档到 `life/archives/daily/`
- 临时文件：清理超过 7 天的临时文件

### 2. 结构验证
- 确保必要目录存在
- 检查 JSON 文件格式正确
- 验证 Markdown 文件可读

### 3. 磁盘空间
- 检查 workspace 大小
- 超过 100MB 警告
- 建议清理策略

### 4. Git 维护
- 检查未提交的更改
- 提醒提交重要更改
- 可选自动提交

## 输出格式

```
## 夜间优化报告 YYYY-MM-DD

### 文件健康
- MEMORY.md: N 行 ✅/⚠️/❌
- 日志文件: N 个
- 已归档: N 个旧日志

### 结构验证
- 目录完整性: ✅
- JSON 格式: ✅/❌
- Markdown 格式: ✅

### 磁盘空间
- 工作区大小: N MB
- 状态: ✅/⚠️

### Git 状态
- 未提交更改: N 个文件
- 建议: [提交建议]

### 执行操作
- [已执行的操作]
```

## 自动操作

以下操作可以自动执行：
- 归档超过 30 天的日志
- 清理临时文件
- 格式化 JSON 文件

需要确认的操作：
- 提交 Git 更改
- 拆分 MEMORY.md
- 删除内容
