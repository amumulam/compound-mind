# 决策日志目录

存放重要决策的记录文件。

## 文件命名

`YYYY-MM-DD-slug.md`

例如：`2026-03-03-use-cron-for-automation.md`

## 决策记录格式

```markdown
# 决策：[标题]

**日期**: YYYY-MM-DD
**决策者**: [谁做的决策]
**触发器**: [什么情况触发了这个决策]

## 背景

[为什么需要做这个决策？遇到了什么问题？]

## 选项

1. **选项 A**: [描述]
   - 优点: ...
   - 缺点: ...

2. **选项 B**: [描述]
   - 优点: ...
   - 缺点: ...

## 决策

选择了：**选项 X**

## 理由

[为什么选这个？主要考虑什么？]

## 影响

- 短期影响: ...
- 长期影响: ...
- 相关决策: [链接到其他相关决策]

## 验证标准

[如何验证这个决策是正确的？什么情况下需要重新考虑？]
```

## 如何触发

1. 在检查点提取时自动识别决策关键词
2. 手动创建决策记录
3. 重要任务完成后主动记录
