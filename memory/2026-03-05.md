## 框架生命周期管理完成

### 创建的脚本

| 脚本 | 功能 |
|------|------|
| update.sh | 更新已安装 agent 的框架配置 |
| uninstall.sh | 卸载框架（保留 memory/ 和 MEMORY.md）|

### 更新内容

**安装脚本**：
- Cron 任务 message 改为英文（国际化支持）

**update.sh**：
- 自动检测 agent 当前使用的 model
- 更新目录结构
- 更新 AGENTS.md 规则
- 更新 Cron 任务 payload

**uninstall.sh**：
- 删除 Cron 任务
- 删除 docs/、life/
- 移除 AGENTS.md 规则
- 保留 memory/ 和 MEMORY.md

### 设计决策

| 决策 | 理由 |
|------|------|
| 更新脚本只支持单选 | 保持简单，符合安装脚本设计 |
| 删除时保留 memory/ 和 MEMORY.md | 这是 agent 的默认资产，不属于框架 |
| 自动检测 model | 避免用户重复输入 |
