# para-system

辅助系统 - 存放定时任务脚本。

## 命名由来

"para-" 来自希腊语，意思是"旁边、辅助、并行"。

para-system = 辅助系统，在后台自动运行，维护记忆系统。

## 已配置的定时任务

| 任务 | 功能 | 频率 | 状态 |
|------|------|------|------|
| 巴巴检查点提取 | 从日志提取关键信息 | 每 6 小时 | ✅ 运行中 |
| 巴巴每周模式提取 | 分析日志发现规律 | 每周日 02:00 | ✅ 运行中 |
| 巴巴每周知识验证 | 检测过时/冲突知识 | 每周日 02:30 | ✅ 运行中 |
| 巴巴每周夜间优化 | 系统健康维护 | 每周日 03:00 | ✅ 运行中 |
